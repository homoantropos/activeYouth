Вітаю! Цей блок буде видалений, зараз він тут аби познайомити вас з структурою сайту.
Вона складатиметься з трьох основних рівнів.
1. Головний рівень.
Доступ до головного рівню відбуватиметься переходом за адресою сайту, ви зараз знаходитесь на ньому.
Тут користувач зможе подивитися новини, уроки з активностями, розклад заходів і рейтинги.
2. Admin рівень.
Доступ до нього можна буде отримати тільки авторизованим користувачам за роутом "адреса сайту/admin".
Зараз ви зможете перейти на адмін рівень за посиланням: admin для логіну емейл santre8@gmail.com пароль test. На реальному сайті його не буде
На адмін рівні авторизовані користувачі зможуть додавати заходи, активності, уроки, результати, редагувати існуючі а також переглядати статистику.
3. FinAdmin рівень.
Доступ до нього можна буде отримати тільки авторизованим користувачам за роутом "адреса сайту/expenses".
Зараз ви зможете перейти на finадмін рівень за посиланням: finadmin для логіну емейл santre8@gmail.com пароль test. На реальному сайті його не буде
На finадмін рівні авторизовані користувачі зможуть керувати витратами по заходам.
